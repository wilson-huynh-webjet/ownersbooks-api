# Owners Books API
A simple REST API for managing books by owner, built with Node.js and Express.

## Requirements
- Node.js v24
- npm

## Development 
### Setup

```bash
npm install
```

## Run dev server

```bash
npm start
```

The server runs at `http://localhost:9000` by default.  

## Endpoints

| Method | Path     | Description    |
|--------|----------|----------------|
| GET    | `/books` | List all owners & books |
